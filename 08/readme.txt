Задание 08

Скорректировать программу 05 так, чтобы доступ к файлу регулировался семафором (SYSTEM V). 

Использование семафора SYSTEM V
При заупуске необходимо ввести аргумент - количесвто случайных чисел, создаваемых дочерним процессом.
Логика работы:
Порождается дочерний процесс, который генерирует случайное число и записыввет в канал.
Родительский процесс читает даынные из канала, выводит их на экран и записывает в файл "numbers" в двоичном формате, при этом во время записи переводит
(если это не так) значение семафора в 1 (закрывает доступ), запрещающая дочернему процессу читать данные из файла, 
после записи значение семафора уменьшается до 0 (доступ открыт)
В это время дочерний процесс, находился в режиме ожидания, пока значение семафора не станет равным 0.
Дочерний процесс считывает данные из файла и выводит их на экран, и переводит значение семафора в 1.
Так повторятся n-ое количесвто раз.
