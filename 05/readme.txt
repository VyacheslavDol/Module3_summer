Изменить программу 03 так, чтобы дочерний процесс  выводил информацию из файла. Если родительский процесс собирается  
изменить файл, то он отправляет сигнал SIGUSR1 (блокировка доступа к  файлу). Когда родительский процесс завершил модификацию файла, 
он  отправляет сигнал SIGUSR2 (разрешение доступа к файлу). Дочерний  процесс отправляет новое число родительскому после того, как прочитал  файл


При заупуске необходимо ввести аргумент - количесвто случайных чисел, создаваемых дочерним процессом.
Логика работы:
Порождается дочерний процесс, который генерирует случайное число и записыввет в канал.
Родительский процесс читает даынные из канала, выводит их на экран и записывает в файл "numbers в двоичном формате, при этом во время записи отправляется сигнал, 
запрещающий дочернему процессу читать данные из файла, после записи отправляется сигнал с разрешением дочернему сигналу на чтение.
Дочерний процесс считывает данные из файла и выводит их на экран
Так повторятся n-ое количесвто раз.
Реакции на сигналы изменены c помощбю функции signal().
