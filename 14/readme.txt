Задание 14 (UDP сокеты)
Реализовать чат для двух клиентов с использованием UDP-сокетов.

Персональный чат организован на основе UDP cокетов. Программы пересылают сообщения друг другу напрямую. IP адреса и номера портов задаются статически.
IP - 127.0.0.1 - локальный хост

Особенности работы программы:
- во время запуска необходимо в разных терминалах указать 1 в другом 2 (./a 1  и ./a 2), таким образом внутри программы назначаются номера портов сокетов.
- для выхода из программы необходимо написать комнаду "exit", при этом собеседнику отправится сообщение о выходе из чата.

Изменения 1:
Для программы sniffer из задания 17 были скорректрованы IP адреса. Теперь процессы будут отправлять сообщения с разных адресов 127.0.0.1 и 127.0.0.5.
