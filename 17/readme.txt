Задание 17 (RAW сокеты)

Написать программу, получающую копии пакетов, предназначенных приложению-серверу из задачи 14 (снифер).
Программа может формировать дамп данных, который нужно вручную проанализировать (просмотр бинарных файлов) и убедиться, 
что это действительно информация от программы-клиента.
Можно дополнить программу так, чтобы она расшифровывала и выводила в файл или на экран полученные сообщения.

Реализована программа сниффер для перехвата пакетов. Реализован простой фильтр с указанием IP адреса и порта назначения. Перехваченные пакеты записываются в файл
с разбором заголовков. На экран также выводится ифнорамция о перехваченном пакете.

Запуск программы необходимо производлить с использованием ROOT прав:
$ sudo ./sniffer <ip_dest> <port_dest> 
Если в ip адресе указать 0, то фильтарыции не будет и будут перехвачены все пакеты с UDP.
Для проверки на примере задания №14:
$ sudo ./sniffer 127.0.0.5 50002 (или 127.0.0.1 50000)

Для корректного выхода из пронграммы рекомендуется использовать CTRL+C (SIGINT) (может сработать немнгновенно), реакция на него выводит процесс из бесконченого цикла -> закрываются все дескрипторы,
освобождается динамическая память.

Работа программы:
Создается RAW сокет UDP, в бесконечном цикле происходит перехват всех UDP пакетов. В функции обработчике происходит фильтрация пакетов, вывод на экран и в файл информации
и перехваченном пакете.
Вывод анализа заголовков в файл корректно работает для загловков IP 20 байт и UDP 8 байт, так как заголовок IP моежт быть больше, в таком случае отображенная информация
в заглолвке UDP будет некорректна.
